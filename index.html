Basic command line commands =========================== 1> ls: This command is used to check the contents of the folder 2> cd: This command is used to change from one directory to another directory 3> touch: touch filename.txt/html/css/js/etc This command is for creating new files in the folders. Wr can create files with different extensions. 4> mkdir: mkdir foldername This command is used to create folder names in the directory. 5> rm: rm filename This command is used to delete the files from the folders or directories. 6> rm -rf: rm -rf foldername: This command is to delete the folders from the directory 7> rm -rf /: This is the most powerful command. This command is used to delete the whole directory. Don't run this command.
